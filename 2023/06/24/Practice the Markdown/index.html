一、 markdown纯文本基本语法

1. 标题

markdown 支持两种标题语法，类setext和atx形式。

1.1 类setext

这种形式用底线形式：‘=’代表最高阶标题，‘-’代表第二阶标题，语句如下（需要顶格写）：

最高阶标题
=========

第二阶标题
-——–

1.2 类atex

这种形式，则是在行首插入1~6个#，对应标题1到6.\# this is H1

## this is H2

2. 字体

markdown使用*或者_作为标记强调字词的符号，强调字词的两边符号相同。
**这是加粗**
__这也是加粗__
*这是倾斜*
_这也是倾斜_
***这是加粗斜体***
~~这是加删除线~~

3. 分割线

你可以在一行中用三个以上的星号、减号、底线建立一个分割线，行内不能有其他东西。
* * *
***
*****
- - -
------

> *这是引用模块中的无序列表

8. 段落与换行

一个markdown是由一个或者多个文本行组成，前后要有一个以上的空行。普通段落不该用空格或者制表符来缩进。两个文字块要用空行来分隔。

三、Markdown 进阶使用

markdown 需要插入一些图片，链接。由于markdown 只关注于纯文本因此需要一些引用。

1.超链接

markdown 两种形式的链接语法：行内式和参考式。不管哪一种，链接文字都要用[方括号]来标记。

  1. 行内式链接
    *要建立行内式链接，只要在方括号后面紧接着圆括号并插入网址链接即可，方括号和圆括号之间不能有空格。如果想加上链接title文字，只要在网址后面用双引号把title包起来即可。
    this is [an example](https://www.baidu.com/ “百度”) inline link.
    效果如下：
    this is an example inline link.
    *如果想要在新页面打开，可以用html语言的a标签代替
    <a href=”https://cn.bing.com/“ target = “_blank”>必应</a>
    效果如下：
    <a href="https://cn.bing.com/" target = "_blank">必应 </a>
  2. 参考式链接
    在链接文字的后面再添加一个方括号并填入应以便是链接的标识。
    This is [an example]<sup>[1]</sup> refernece-style link.
    接着在文件的任意处，你可以把这个标记的链接内容定义出来。
    [1]: \https://cn.bing.com/ "bing”
    效果如下:
    This is [an example]<sup>[1]</sup> refernece-style link.
    [1]: https://cn.bing.com/ “bing”

2.自动链接

处理网址和电子邮箱，只要用尖括号包起来，markdown会自动把它转换成链接
https://cn.bing.com/
